# Hardware Module

This folder contains all the firmware and related hardware files for the V2X BOT project.

## Directory Structure

- **src/**: Source code for all ESP32-CAM applications.
  - **camera_web_server/**: Original base camera web server implementation.
  - **esp32cam_app1/**: Primary ESP32 application.
  - **esp32cam_app2/**: Secondary alternative ESP32 app/firmware.
- **firmware/**: Pre-compiled binary artifacts ready for flashing.
- **docs/**: Schematics, wiring diagrams, and hardware-specific documentation.
